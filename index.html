SPACE CSS v0.1

Space CSS is a framework that helps you create amazing projects developing responsive. It's made to work on dynamic sizes and help your development to be faster and easier. This framework is built using Sass and you can get started with precompiled CSS or build your own source code. To use Space CSS by the precompiled CSS, all you need to do is download the file "space.css" and import on your HTML file or CSS.

Otherwise if you want to build your own code you can change the files and after compile the Sass file space.scss.

Space CSS is an open souce project, and you can find the code on GitHub.

Grid

The grid system is based on 12 columns (one column = 100% width/12), to use the grid you just need to put the class .grid-DEVICE-COLUMN_NUMBER where DEVICE is the size of the device ('xs' to extra small devices like phones, 's' to small devices like tablets, 'm' to medium devices like desktops or notebooks and 'l' to large devices like large desktops) and COLUMN_NUMBER is the number of columns.

You can use .container class to put the content inside a div with margins and a calculated gutter padding. Otherwise the .container-full class is used to put the content inside a full width div. Use .row class to put your content in one line (if the size of the content is smaller than your window width).

Semantic Grid

You can improve your grid semantic using the semantic grid system. All you have to do is to extend the grid class in your semantic class and replace COLUMNS for your column number.

Usage

Use "xs" to extend the class on extra small devices. @extend .grid-xs-COLUMNS;

Use "s" to extend the class on small devices. @extend .grid-s-COLUMNS;

Use "m" to extend the class on medium devices. @extend .grid-m-COLUMNS;

Use "l" to extend the class on large devices. @extend .grid-l-COLUMNS;

To extend the grid class to all breakpoint sizes, use "all" where you put the device size. @extend .grid-all-COLUMNS;

Example

.full{ @extend .grid-all-12; }

.one-half{ @extend .grid-all-6; }

.one-third{ @extend .grid-all-4; }

.one-quarter{ @extend .grid-all-3; }
